# 🛒 Mini Shop Full Stack App

A full-stack e-commerce system built with:

- Mobile App (Expo React Native)
- Backend API (Fastify + TypeScript)
- Admin Dashboard (React + Tailwind)
- Database & Auth (Supabase)

---

## 📁 Project Structure

mini-shop/
├── mobile/
├── backend/
├── dashboard/

---

## ⚙️ Tech Stack

- React Native (Expo SDK)
- Redux Toolkit + RTK Query
- Fastify + Node.js
- TypeScript (Full project)
- Supabase (Auth + DB + Storage)
- React + Tailwind CSS (Dashboard)

---

## 🚀 How to Run

### 📱 Mobile App

```bash
cd mini-shop
npm install
npx expo start

### 📱 back end

cd mini-shopBackEnd
npm install
npm run dev

### 📱 dashboard
cd shop-dashboard
npm install
npm run dev
